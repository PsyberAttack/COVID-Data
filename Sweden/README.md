## Sweden COVID Data

Includes data from the Swedish Public Health Agency, (Folkhalsomyndigheten).

*** THIS  DATASET IS IN PROGRESS. DAILY UPDATES NOW OCCURRING***
 
 
### Files:

**Counties** contains time series data for each Swedish county


### Data Sources:
